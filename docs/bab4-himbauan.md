# BAB IV  
# HIMBAUAN BAGI PRAJURIT TNI AU

Berdasarkan telegram Kepala Staf Angkatan Udara:

1. Gunakan jaringan resmi untuk dokumen rahasia.
2. Jangan mempublikasikan dokumen rahasia tanpa izin.
3. Batasi akses sistem hanya untuk pengguna terdaftar.
4. Gunakan antivirus resmi dan aktifkan 2FA.
5. Abaikan tautan mencurigakan.
6. Logout setelah penggunaan aplikasi chat.
7. Ikuti protap penghancuran dokumen.
8. Laporkan pelanggaran melalui jalur komando.
